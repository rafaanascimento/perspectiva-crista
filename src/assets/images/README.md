# Imagens de posts

Coloque aqui as imagens locais utilizadas no conteúdo.

**Padrões recomendados**
- Formato: `.jpg` ou `.webp`
- Largura mínima: 1600px
- Proporção sugerida: 16:9
- Nome do arquivo: `YYYY-MM-DD-titulo-curto.jpg`

> Este repositório não inclui imagens reais. Adicione-as manualmente quando necessário.
