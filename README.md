# Perspectiva Cristã

Blog editorial cristão focado em artigos, reflexões e colunas, com publicação estática e conteúdo como protagonista.

## Objetivo

Criar uma base simples, sustentável e orientada a conteúdo para textos autorais sobre fé, reflexão e propósito.

## Stack

- [Astro](https://astro.build/)
- Markdown
- CSS moderno (sem frameworks JS)

## Organização de conteúdo

- `src/content/posts`: artigos em Markdown com frontmatter obrigatório.
- `src/data/authors.json`: cadastro de autores.
- `public/images`: imagens públicas do site.
- `public/images/authors`: fotos de autores.

## Uso pessoal/editorial

Projeto de uso pessoal e editorial, pensado para evolução contínua de conteúdo.
